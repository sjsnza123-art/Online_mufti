# Online Mufti App

**آن لائن مفتی** — A Flutter app for Islamic guidance (فتاویٰ، خطبات، رابطہ)

## Features
- سوال و جواب کا نظام
- فتاویٰ آرکائیو
- خطبات و دروس
- رابطہ فارم

## How to Run
```bash
flutter pub get
flutter run
```

GitHub Actions workflow automatically builds APK and AAB on every push.
